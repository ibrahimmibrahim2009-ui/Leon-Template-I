# Leon Template I Project
## A fully responsive website with comfortable design colors and layout. 
## Sections
### The web is devided into many sections: 
1. Header: It includes a fontawesome icon and droplist. The droplist is designed with hover property to display the droplist links that are directed to the web sections. 
2. Hero-section: It includes a title and subtitle for the web with a landscape background.
3. Features: It includes three boxs, each contain an icon, heading, and paragraph.
4. Services, Portfolio, About, Contact sections: These sections have main similarities in their layout. Some images are designed with before and after Pseudo Elements.
